Project Overview
What We Built

This is a three-page static website deployed on GitHub Pages for Computer Networking Project 1. The goal was to get a site live and then document the networking infrastructure behind it - DNS resolution, IP addressing, HTTP/HTTPS headers, and the security features that GitHub Pages includes by default.

The stack is intentionally simple: plain HTML and CSS, no build tools or frameworks. That kept the focus on the networking side rather than frontend tooling.
